<objective>
## Goal
Add two new top-of-page status bars that mirror the existing urgent-tasks bar: a "My Day" bar listing tasks scheduled or due today, and a "Neglected" bar listing the top 5 stale tasks. Both render alongside (below) the urgent bar, hide when empty, and open the relevant task flyout when a chip is clicked.

## Purpose
The urgent bar is a high-signal at-a-glance surface that the user already loves. Replicating that affordance for "what's on the agenda today" and "what's slipping" raises the visibility of two commonly-missed task categories without forcing the user to switch panels. This is a top-priority insertion ahead of Phase 3 because the user requested it as immediate.

## Output
- New `#myDayBar` and `#neglectedBar` DOM elements in `index.html`
- Mirrored CSS classes (`.myday-bar`, `.myday-chip`, `.neglected-bar`, `.neglected-chip`) reusing the urgent-bar visual language with a category-distinguishing accent
- New `getMyDayTopBar()` and `getNeglectedTasks()` methods on `TaskManager`
- New `renderMyDayBar()` and `renderNeglectedBar()` methods on `UIController`, both wired into `renderAll()`
- Tests covering the two new selector methods
</objective>

## AC-1: My Day bar shows today's tasks

```gherkin
Given the user has tasks where myDayDate equals today's local date
  And/or tasks where dueDate equals today's local date
  And those tasks are not completed
```

## AC-2: Neglected bar shows top 5 stale tasks

```gherkin

  And it contains chips for at most the 5 oldest tasks whose updatedAt is older than N days
  And chips are sorted updatedAt ascending (oldest first)
  And clicking a chip opens that task's flyout
  And when fewer than 1 task qualifies the bar is hidden
```
