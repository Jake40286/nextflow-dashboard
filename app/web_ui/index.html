                                    How NextFlow works

                                    1. Capture new ideas in the Inbox using the quick add bar.
                                    2. Clarify each item by editing it—set the desired status, context, due dates, and project.
                                    3. Organize tasks into Next Actions, Waiting For, or Someday by dragging or editing.
                                    4. Review projects regularly: ensure every active project has a clear outcome and at least one next action.
                                    5. Engage with the active tab that matters now—filter by context, run the Pomodoro timer, and check off progress.

                                    Need more detail? Use Export JSON for a full backup, or Export .md to review active tasks in Obsidian.
